Easily convert select elements into beautiful lists.

Easy to use... No special Clases or Functions, just 100% reason to remember the name.

Using it.

Code:

<select class='mySelect' style='width:100%;'>
 <option value="volvo">Volvo</option>
 <option value="saab">Saab</option>
 <option value="opel">Opel</option>
 <option value="audi">Audi</option>
</select>



$('.mySelect').listify(
   function(e){
      alert(e.target.innerHTML)
   }
)
